
1. 总结下命令用法
add, commit, push, pull 基础操作，从working tree 到 index 到 repository 到 remote，在拉取到local

branch 分支查询，用来查看branch信息
remote 管理本地与远程分支的联系
checkout 在新版本中，功能被 switch + restore 瓜分
switch 新建、切换分支
restore 恢复文件状态



diff 查询差异

status 查询 repository 状态

stash 临时存储working tree

rm 删除文件并提交

cherry-pick 少量提交的合并操作
merge 分支的合并操作，以树状log记录
rebase 分支的合并操作，以线性记录，用来梳理本地repository的提交历史

log     提交日志，用来查找commit的哈希
reflog  引用日志，记录HEAD指针，branch的引用，可以恢复 rebase，revert，reset等操作造成的错误

reset   三种参数，主要作用是移动 HEAD 指针，实现commit回退，没有提交记录的
revert  撤销 commit 的提交内容，是有提交记录的
