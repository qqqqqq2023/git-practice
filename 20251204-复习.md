基础命令
add 将文件设置为已追踪，从working tree放到index
commit 提交文件到repository
pull 从远程拉取到local
push 从local 推送到 remote -u 设置remote
日志查看，自定义命令 one last graph
git rm 删除文件，物理删除，提交commit
stash 暂存working tree + index
status 查看local状态


分支管理
branch 查询分支信息
    -v local
    -r remote
    -vv local:remote
    -a local+remote
    -d 删除分支
    -D force删除

switch 分支创建、切换
    -c 创建
    -d 分离HEAD指针到指定commit


文件差异
git diff 


合并
merge 
cherry-pick
rebase

数据恢复
restore
    -S index
    -W working tree

reset 移动HEAD指针
    --soft      working tree + index 都保留
    --mixed     working tree 保留
    --hard      working tree + index 都不保留

revert 对指定commit生成逆向commit以完成回撤

reflog  引用日志，查询HEAD指针移动的历史，用来移动HEAD指针，
